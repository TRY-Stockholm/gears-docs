# Gears Docs

Implementation guides for Made People's Gears packages, optimized for consumption by AI coding assistants working on client projects outside the monorepo.

Deployed: TBD (Vercel).

## Local development

```bash
bun install
bun run dev
```

Open http://localhost:3000.

## Stack

- Next.js App Router
- Fumadocs UI + MDX
- Tailwind v4
- Bun
